# 附則（NEarthサーバー）
この文書（以下「本附則」）は、[Nekozouneko Server 利用規約](/terms)の附則とします。
## 第一条（本文における定義）
|用語|定義|
|-|-|
|Lands|プラグインの一種、主に国家、土地及び戦争の管理を行います。|
|集団|Landsにおける次のものの総称、集落、村、町、都市、大都市、連邦、国及び帝国|
|識別名|集団名及び役職名を含む、Landsのシステム上第三者が閲覧可能な場所に明記する名前全般|
|オーナー|集団のオーナー役割に指定されるプレイヤー|
|所属者|集団に所属するプレイヤー|
|集団内規則|オーナーと各所属者との間に適用される所属条件及び禁止事項等の各種定め|
|アースマップ|地球の地理データを一定比率でワールドに再現し、その現実と仮想との繋がりがあることを魅力とするマップ全般|
|敵対機能|Landsにより、システム的に認められる外交及び戦争関係の機能（集団を敵対としてマークしたり、戦争を実施する機能を含みます。）|
|攻撃側|戦争を行う際、宣戦布告を先に送信した側の集団|
|防衛側|戦争を行う際、宣戦布告を受け取って承認した側の集団|
|エリア|Landsにより、自由な範囲でより小さく分割された区画及び領土全体の空間|
## 第二条（オーナーの責任）
1. [オーナー](#第一条-本文における定義)は、[集団](#第一条-本文における定義)の運営に係る一切の責任を負うものとします。
2. オーナーは、集団内の事情及び[所属者](#第一条-本文における定義)に、当サーバーが定める各種定めを厳守させる義務を負うものとします。
## 第三条（集団内規則）
1. オーナーは、個々の形式に基づき[集団内規則](#第一条-本文における定義)を制定することができるものとします。
2. オーナーは、集団内規則を制定又は改正するとき、当サーバーが定める各種定めに抵触及び矛盾しないものとします。
3. オーナーは、所属者に対して現行の集団内規則を十分に周知させるものとします。
## 第四条（禁止事項）
下記のいずれかに該当する行為をしてはなりません。
1. [Nekozouneko Server 利用規約第三条（禁止事項）](/terms#第三条-禁止事項)第17項、第18項、第19項及び第20項に準ずる識別名又はなりすましの意図をもって他集団の識別名と著しく類似する識別名を使用する行為。
2. 同一人物が4つ以上のアカウントを使用する行為
3. [アースマップ](#第一条-本文における定義)の魅力を著しく低下させる程度にマップを改変する行為。
4. 集団の周辺を囲うように土地を確保するなど、[敵対機能](#第一条-本文における定義)を使用せず他の集団に損害を与える行為。
## 第五条（敵対機能）
1. オーナーは、戦争を実施する機能（以下「戦争機能」）を用いて、他集団と互いに合意の上、戦争を実施することができます。
2. 攻撃側の所属者及び防衛側の所属者は、戦争が行われている間、次に掲げる行為を行うことができます。
    1. 攻撃側の所属者及び防衛側の所属者が互いに行うプレイヤーの殺害行為又は体力若しくは防具に損害を与える行為
    2. 前項によるプレイヤーの殺害時にドロップしたアイテム及び経験値等を獲得する行為
## 第六条（処罰及び対応方針）
1. 当サーバーは、[第三条（集団内規則）](#第三条-集団内規則)第2項に該当する集団内規則を制定していると判断した集団に対して、オーナーに相当期間定めてなした勧告後も状況が是正されない場合、集団の削除を強制することができるものとします。
2. 当サーバーは、[第四条（禁止事項）](#第四条-禁止事項)第3項に該当すると判断した改変行為に対して、その所有者に相当期間定めてなした勧告後も状況が是正されない場合、強制的に再生成又はロールバックをすることができるものとします。
3. 当サーバーは、集団が[Nekozouneko Server 利用規約第三条（禁止事項）](/terms#第三条-禁止事項)又は本附則のいずれかの条項に抵触したと判断した場合、集団の削除を強制することができるものとします。
## 第七条（免責事項）
1. 当サーバーは、集団が[エリア](#第一条-本文における定義)において自ら設定した各種権限に起因して生じた損害について、一切の責任を負わないものとします。
2. 当サーバーは、[第三条（集団内規則）](#第三条-集団内規則)に基づき制定された集団内規則に起因して生じた損害及び紛争等について、一切の責任を負わないものとします。
3. 当グループに所属の者が、当サーバーの運営を目的として導入したアカウントに関して、[第四条（禁止事項）](#第四条-禁止事項)第2項はその限りではありません。
4. 当サーバーは、敵対機能によって生じたアイテムの損失について、一切の責任を負わないものとします。
5. 当サーバーは、Lands以外の媒体により主張された土地について、一切の責任を負わないものとします。
