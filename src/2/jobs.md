1. `/jobs browse`を実行すると下記のようなGUIが開きます。

2. 就きたい職業を`右クリック`すると職業に就くことができます。
## 仕事の報酬
各仕事の仕事量に対して貰える報酬は、サーバーによって調整されています。主には、下記の値を使って計算されます。
* ベース値（お金・ポイント・EXP）
  * 職業によってアイテムごとに決められています。
* 職業のレベル（最大200レベル）
* 就いている職業数
* 報酬調整パラメータ
  * 職業ごとのバランス調整用です。
### 計算方法
職業ごとのレベルが高ければ高いほど、報酬は多くなっていきます。同時に就いている職業が多ければ多いほど、報酬単価は少なくなります。
::: warning
職業のレベルと就いている職業数が同じだったとしても、必ずしも全サーバーで同じように報酬量が計算されるわけではありません。また、報酬調整パラメータだけで調整できない不規則的なバランス調整によって、ベース値も変わっていることがあります。
:::
```txt{4}
レベルによる増加額T = ベース値*(職業のレベル-1)*0.01
レベルによる増加額の適用後額S = ベース値 + T
職業数による報酬減少係数D = 1-(就いている職業数-1)*0.05
報酬量 = S * D * 報酬調整パラメータ
```
