# QuickShop
自分だけのショップを建てることができるプラグインです。アイテムを売ってお金を稼ぐことができます。

## ショップの作り方
物を売ってお金を稼ぎたい場合は、ショップを立ててプレイヤーに買ってもらいましょう。
1. チェストを設置する
2. 売りたいアイテムを持ってチェストを`左クリック`する
<br>左クリックして次のようなメッセージが表示されるのが正常です。
![](/images/quickshop/sell_price.png)
3. チャットに**1つあたりの値段**を入力して送信する
4. 出品完了！
<br>チェストの看板が自動で付いていたら成功です。

## ショップでの買い方
1. ショップのチェストに付いているチェストを`左クリック`する
<br>ショップ情報と次のようなメッセージが表示されるのが正常です。
![](/images/quickshop/buy_amount.png)
2. チャットに購入したい個数を入力して送信する
:::tip
ステップ1で表示される`〇〇個 購入できます`の個数は、自分の持っている残高または在庫がある限りの実際に購入できる個数の上限が表示されます。
:::
3. 購入完了！
<br>購入成功のメッセージと購入した個数、消費した残高が表示されれば成功です。

## 購入されたら...
ショップでプレイヤーから物が買われたとき、次のようなメッセージが表示されます。
![](/images/quickshop/sell_bought.png)
::: tip
ショップでの収益は、すぐに販売主の残高に直接振り込まれます。
:::
