# 集団の作り方
::: warning
集団を作るには設立費用がかかる場合があります。各サーバーの設定によって異なるため、各サーバーの説明をご参照ください。
:::
集団を作るには`/lands create <集団名>`コマンドを使用します。このとき、今自分がいるチャンクが集団のものとして自動的に確保されます。そこが集団のスポーンになります。
## チャンクの境界を見る方法
### Java版の場合
`F3`キーと`G`キーを同時押し（`F3+G`）することで、チャンクの境界線表示を切り替えることができます。
### 統合版の場合（Java版でも対応）
サーバーに参加すると下記のようなアイテムが付与されます。左から二番目の羽`情報ツール`を持って、地面を右クリックすると、緑色のスモークで周りにチャンクの境界が表示されます。
