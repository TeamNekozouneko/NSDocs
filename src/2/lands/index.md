# Lands
町や国を建てることができるプラグインです。土地の管理機能や戦争機能が標準で搭載されています。
::: tip
集団を設立する費用や、維持費などはサーバーによって異なります。詳しくは各サーバーの説明をご参照ください。
:::
## 概念
Landsは`集団`と`国`という2種類の扱いがあります。
### プレイヤーの成り行き
1. プレイヤーは、プレイヤーの単なる集まりとして`集団`を作成します。
<br>集団単位で土地を取ることができるため、お金を稼ぎながら土地を広げていきます。
2. 徐々に`集団`は様々な呼び方で呼ばれるようになります。
<br>住民の人数や、銀行残高に応じて下記のように呼ばれます。
::: info
①集落 → ②村 → ③街 → ④都市 → ⑤大都市
:::
3. `集団`は次第に人脈を広げ、大都市と呼ばれるようになります。
<br>他の集団との交流もあり、時には対立も生まれます。
4. `集団`は別の集団と手を組み、集団の集まり`国`を設立します。
<br>国もまた所属する集団の数や、住民の人数、銀行残高に応じて下記のように呼ばれます。
::: info
①連邦 → ②国 → ③帝国
:::
