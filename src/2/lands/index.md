# Lands
町や国を建てることができるプラグインです。土地の管理機能や戦争機能が標準で搭載されています。
::: tip
集団を設立する費用や、維持費などはサーバーによって異なります。詳しくは各サーバーの説明をご参照ください。
:::
## 概念
Landsは`集団`と`国`という2種類の扱いがあります。
### プレイヤーの成り行き
1. プレイヤーは、プレイヤーの単なる集まりとして`集団`を作成します。
<br>集団単位で土地を取ることができるため、お金を稼ぎながら土地を広げていきます。
2. 徐々に`集団`は様々な呼び方で呼ばれるようになります。
<br>住民の人数や、銀行残高に応じて下記のように呼ばれます。
::: info
①集落 → ②村 → ③町 → ④都市 → ⑤大都市
:::
3. `集団`は次第に人脈を広げ、大都市と呼ばれるようになります。
<br>他の集団との交流もあり、時には対立も生まれます。
4. `集団`は別の集団と手を組み、集団の集まり`国`を設立します。
<br>国もまた所属する集団の数や、住民の人数、銀行残高に応じて下記のように呼ばれます。
::: info
①連邦 → ②国 → ③帝国
:::

## コマンド
::: tip コマンドに含まれる記号の意味（例）
* `<集団名>`：集団名を入力しなければならない
* `[集団名]`：集団名を入力するかは任意
* `<集団名 | here>`：集団名**又は**hereを入力しなければならない
* `[radius | auto | fill]`：radius、auto又はfillを入力するかは任意
* `<(1)radius: 半径>`：左から1個目の入力がradiusの場合は、半径を入力しなければならない
* `[(1)radius: 半径]`：左から1個目の入力がradiusの場合は、半径を入力するかは任意
* `*`：あり得る候補すべて
:::

### 一般的なコマンド
`/lands`  
LandsのGUIメニューを開く。

`/lands menu [集団名|here]`  
指定した集団名又は今いる場所のLandsのGUIメニューを開く。

`/lands help [ページ]`  
Landsのコマンド一覧を表示する。

`/lands claim help`  
土地の宣言に関するコマンド一覧を表示する。

`/lands create <集団名>`  
集団を作成する。今いるチャンクがホームチャンクになります。

`/lands createcamp <キャンプ名>`  
今いるチャンクに一時的なキャンプを作成する。

`/lands delete <集団名 | here>`  
集団を削除する。hereを指定したとき、サブエリアに立っている場合はサブエリアが削除されます。

`/lands selection`  
選択モードを切り替えます。左クリックと右クリックで範囲の角とその対角線側にある角を選択します。選択し終わると、その範囲をどうするかを次のコマンドで決定します。
* `/lands claim`：範囲を集団のチャンクとして宣言する。
* `/lands assign <サブエリア名>`：範囲をサブエリアとして設定する。
* `/lands unclaim`：範囲を集団のチャンクから削除する。

`/lands claim [radius | auto | fill] <(1)radius: 半径>`  
radius、auto又はfillのいずれも入力せずに実行した場合、今いるチャンクを集団のチャンクとして宣言します。
* `radius`：今いるチャンクの周囲を集団のチャンクとして宣言します。
* `auto`：自動取得モードを切り替えます。（歩いた場所を自動で集団のチャンクとして宣言する）
* `fill`：現時点で集団が宣言しているチャンクで囲まれたチャンクを埋めるようにすべて宣言します。

`/lands claimlist`  
集団が宣言しているチャンク一覧を表示します。

`/lands unclaim [radius | auto] <(1)radius: 半径>`   
radius又はautoのいずれも入力せずに実行した場合、今いるチャンクを集団のチャンクから削除します。
* `radius`：今いるチャンクの周辺を集団のチャンクから削除します。
* `auto`：自動削除モードを切り替えます。（歩いた場所を自動で集団のチャンクから削除する）

`/lands merge <集団名>`  
入力した集団のチャンクを、自身の集団のチャンクと結合します。

`/lands relations`  
集団の関係性を管理するGUIを開きます。

`/lands deposit <金額> [集団名]`  
集団の銀行にお金を預け入れます。

`/lands withdraw <金額> [集団名]`  
集団の銀行からお金を引き出します。

`/lands balance [集団名]`  
集団の銀行残高を表示します。

`/lands trust <プレイヤー名> [集団名 | サブエリア名]`  
集団または特定のサブエリアに対して、プレイヤーを信頼（≒招待）します。集団名又はサブエリア名のいずれも入力せずに実行した場合、プレイヤーは集団全体から信頼されるようになります。

`/lands untrust <プレイヤー名> [集団名 | サブエリア名]`  
集団または特定のサブエリアに対して、プレイヤーを信頼しなくなります（≒追放）。集団名又はサブエリア名のいずれも入力せずに実行した場合、プレイヤーは集団全体から信頼されなくなります。

`/lands ban <プレイヤー名> [集団名 | サブエリア名] [サイレント]`  
プレイヤーを集団又はサブエリアからBANします。集団名又はサブエリア名のいずれも入力せずに実行した場合、プレイヤーは集団全体からBANされます。また、サイレントに`true`を指定すると、プレイヤーにBANしたことを通知しません。

`/lands unban <プレイヤー名> [集団名 | サブエリア名]`  
集団又はサブエリアのプレイヤーのBANを解除します。集団名又はサブエリア名のいずれも入力せずに実行した場合、プレイヤーは集団全体からのBANが解除されます。

`/lands setrole <プレイヤー名> <集団名 | サブエリア名 | *> <役職名>`  
集団又はサブエリアにおけるプレイヤーの役職を設定します。サブエリア名に`*`を入力すると、プレイヤーはすべてのサブエリアでその役職が付与されます。

`/lands setowner <プレイヤー名>`  
集団のオーナーを設定します。

`/lands accept <集団名>`  
集団からの招待（trust）や、所有権リクエストを承諾します。

`/lands deny <集団名>`  
集団からの招待（trust）を拒否します。

`/lands level`  
集団のレベルの進捗状況を確認するGUIを開きます。

`/lands invites`  
現在受け取った招待や所有権リクエストのすべてを確認するGUIを開きます。

`/lands leave <集団名 | here>`  
集団から抜けます。`here`を入力した場合、今いるサブエリアから抜けます。

`/lands chat [集団名 | メッセージ] <(1)集団名: メッセージ>`  
集団内チャットにメッセージを送信します。`/lands chat`だけ入力した場合、集団内チャットを切り替えます。（集団内チャットが有効の場合、通常通りチャットすると集団内だけに送信されます。）集団名を指定した場合、その次にメッセージを入力してください。集団名を指定しない場合は、そのままコマンドにメッセージを入力してください。

`/lands taxes`  
今後の集団ごとに定められた税金の支払額を表示します。

`/lands upkeep`  
集団の維持費と次回の徴収日を確認します。

`/lands info [集団名]`  
集団の情報を表示します。集団名をいずれも入力せずに実行した場合、今いるチャンクの集団情報を表示します。

`/lands player <プレイヤー名>`  
プレイヤーの情報を表示します。

`/lands map`  
今いるチャンクの周辺の宣言状況を表示するGUIを開きます。

`/lands rename <新たな集団名>`  
集団の名前を変更します。

`/lands setspawn`  
集団のホームスポーンを設定します。

`/lands spawn [集団名]`  
集団のホームスポーンにテレポートします。

`/lands list`  
サーバーにあるすべての集団一覧を表示するGUIを開きます。

`/lands storage`  
集団内で共有の仮想チェストを開きます。権限を持つプレイヤーは、受信箱から誰が何を出し入れしたかの履歴を確認することができます。

`/lands view`  
集団のボーダーをパーティクルで可視化して表示します。

`/lands unstuck`  
最寄りの宣言されていないチャンクへテレポートします。集団の中に閉じ込められてしまった場合は、このコマンドを使って脱出することができます。

`/lands rent`  
レンタルに関するコマンドです。

### 国に関するコマンド
`/nations menu`  
国のGUIメニューを開きます。

`/nations create [国名]`  
国を作成します。

`/nations delete`  
国を削除します。

`/nations rename <新たな国名>`  
国名を変更します。

`/nations setcapital <集団名>`  
集団を国の首都に設定します。

`/nations relations`  
国の関係性を管理するGUIを開きます。

`/nations trust <集団名>`  
国に集団を招待します。

`/nations untrust <集団名>`  
国から集団を削除します。

`/nations accept <国名>`  
国からの加盟の招待を受理します。

`/nations deny <国名>`  
国からの加盟の招待を拒否します。

`/nations leave`  
国の加盟から抜けます。

`/nations spawn`  
国の首都にテレポートします。

`/nations chat [国名 | メッセージ] <(1)国名: メッセージ>`  
国内チャットにメッセージを送信します。`/nations chat`だけ入力した場合、国内チャットを切り替えます。（国内チャットが有効の場合、通常通りチャットすると国内だけに送信されます。）国名を指定した場合、その次にメッセージを入力してください。国名を指定しない場合は、そのままコマンドにメッセージを入力してください。

`/nations info [国名]`  
国の情報を表示します。国名をいずれも入力せずに実行した場合、今いるチャンクの国情報を表示します。

`/nations level`  
国のレベルの進捗状況を確認するGUIを開きます。

`/nations list`  
サーバーにあるすべての国一覧を表示するGUIを開きます。

### 戦争に関するコマンド
`/wars menu`  
戦争のGUIメニューを開く。

`/wars declare <集団名 | 国名>`  
集団又は国に対して宣戦布告を送信します。

`/wars deny`  
宣戦布告を拒否します。

`/wars info`  
現在進行中の戦争又は今後予定されている戦争の情報を表示します。

`/wars list`  
サーバーで行われているすべての戦争を表示します。
