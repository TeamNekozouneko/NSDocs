# 銀行にお金を入れる
集団には、集団が持つ残高（銀行残高）が存在します。基本的に、維持費や土地を確保するときの費用は銀行残高から差し引かれます。住民は`/lands deposit <金額>`でお金を預けることができます。
::: tip
コマンドに入力する`<金額>`は、数字のみで単位を含める必要はありません。
* ✅ `/lands deposit 20`
* ❌ `/lands deposit 10G`
* ❌ `/lands deposit 30円`
:::
## お金を引き出す
預け入れた残高は`/lands withdraw <金額>`で引き出すことができます。
## 銀行残高を見る
集団の今預けられている残高の総額は`/lands balance`コマンドを使用します。
