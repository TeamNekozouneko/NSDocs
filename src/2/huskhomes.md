# HuskHomes
サーバーによって使えないコマンドがあります。サーバーによるコマンドの実装状況を次のように表記しています。
<Badge type="danger" text="使用できないサーバー名" /><Badge type="warning" text="一部の使用法ができないサーバー名" />
## ホームを設定しよう！
我が家に帰りたくありませんか？すぐに設定しましょう！

`/home [ホーム名]`: 自分のホームの1つにテレポートする
::: info 説明
名前を省略すると、デフォルトのホームへ飛びます。「帰宅！」と思ったらまずはコレ。
:::

`/homes [ページ]`: 自分のホームのリストを表示する
::: info 説明
「あの拠点の名前、なんだっけ？」設定したホームを一覧で確認できます。
:::

`/sethome [ホーム名]`: 名前を付けて新しいホームを設定する
::: info 説明
「ここをキャンプ地とする！」現在地をホームとして保存します。名前を付ければ複数管理も可能です。
:::

`/delhome [name]`: 以前設定したホームを削除する
::: info 説明
使わなくなった拠点は整理整頓。スロットを空けて新しい冒険へ！
:::

`/delhome all [confirm]`: 自分のホームをすべて削除する
::: info 説明
**注意！** 全てのホーム設定を消去します。心機一転したい時以外は使わないように。
:::

### ホーム情報の編集
`/edithome <ホーム名>`: ホームの情報を表示・編集する
::: info 説明
ホームの詳細メニューを開きます。アイコンや細かい設定はここから。
:::

`/edithome <ホーム名> rename <新しいホーム名>`: ホームの名前を変更する
::: info 説明
「仮拠点」から「本拠点」へ。ホームの名前を後から変更できます。
:::

`/edithome <ホーム名> description <説明文>`: ホームの説明文を設定する
::: info 説明
ホームリストに表示されるメモを残せます。「鉱石倉庫」「交易所」など書いておくと便利！
:::

`/edithome <ホーム名> relocate`: ホームを現在地に再設定する
::: info 説明
リフォームや引っ越しに便利。ホームの名前などの設定はそのままに、ワープ地点だけを今いる場所に更新します。
:::

`/edithome <ホーム名> privacy [public|private]`: ホームのプライバシー設定（公開/非公開）を行う
::: info 説明
自慢の建築をみんなに見せたい？それとも秘密基地にする？公開設定は慎重に！
:::

## 他人の家に行きたい？その家が公開なら、できるぜ！
`/phome [<所有者の名前.ホーム名>]`: 公開ホームにテレポートする
::: info 説明
サーバー内の誰かが公開しているホームへ遊びに行けます。観光してみましょう！
:::

`/phomelist [ページ]`: 公開ホームのリストを表示する
::: info 説明
現在公開されている全てのホームを一覧表示します。素敵な建築が見つかるかも？
:::

## 始まりの大地へ行こう。
`/spawn`: スポーン地点にテレポートする <Badge type="danger" text="ライフ" />
::: info 説明
全ての始まりの場所、初期リスポーン地点へ戻ります。
:::

## 他の人のところにテレポートしてみよう。
`/tpa <プレイヤー名>`: 他のオンラインユーザーにテレポートリクエストを送信する
::: info 説明
「そっち行っていい？」相手に許可を取ってからテレポートします。マナー良く遊びましょう。
:::

`/tpahere <プレイヤー名>`: 他のオンラインユーザーに自分のもとへテレポートするようリクエストを送信する
::: info 説明
「こっち来て見てよ！」相手を自分の場所に呼び出したい時はこちら。
:::

`/tpaccept`: 最後に受信したテレポートリクエストを承諾する
::: info 説明
誰かからのテレポート依頼を許可します。「いいよ、おいで！」
:::

`/tpaccept [プレイヤー名]`: 特定のユーザーからのテレポートリクエストを承諾する
::: info 説明
複数のリクエストが来ている時に、特定の人だけを許可できます。
:::

`/tpdeny`: 最後に受信したテレポートリクエストを拒否する
::: info 説明
今は忙しい？そんな時はこのコマンドでお断りしましょう。
:::

`/tpdeny [プレイヤー名]`: 特定のユーザーからのテレポートリクエストを拒否する
::: info 説明
特定の人からのリクエストだけを拒否します。
:::

`/tpignore`: テレポートリクエストの受信（無視）を切り替える
::: info 説明
建築に集中したい時など、通知自体をオフにする「取り込み中」モードです。
:::

## 未開拓地点へとレッツゴー！
`/rtp`: 現在のワールドのランダムな場所（未開拓地）にテレポートする <Badge type="danger" text="NEarth（代替：/brtp）" />
::: info 説明
冒険の始まりはここから！誰もいない土地へランダムワープします。資源採取や新居探しにどうぞ。
:::

## 戻ろう、あの場所へ。
`/back`: 最後の位置に戻る（使い方は下記参照）<Badge type="danger" text="NEarth" />
* 最後にテレポートした場所に戻る
::: info 説明
「間違えてワープしちゃった！」という時も安心。
:::
* 最後に死亡した場所に戻る
::: info 説明
冒険中の不慮の事故...でも大丈夫。散らばったアイテムを回収しに戻りましょう。
:::
