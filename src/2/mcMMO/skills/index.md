# スキル
合計16種類のスキルがあり、それぞれのスキルにはアビリティがあります。左のタブからそれぞれのスキル内容を確認できます。
## スキルとは
スキルごとに決まっている特定の動作を行うことで、それぞれEXPを獲得できます。特定のEXP数を上回るとスキルレベルが上がります。
## サブスキル（パッシブスキル）
パッシブスキルは、各スキルごとに決まっているサブスキルの内、スキルの発動を**宣言しなくても自動的に発動**するスキルです。サブスキル自体は、各スキルのレベルで内容が変化していきます。
## サブスキル（アクティブスキル）
アクティブスキルは、各スキルごとに決まっているサブスキルの内、スキルの発動を**宣言することで発動**するスキルです。
