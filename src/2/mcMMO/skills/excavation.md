# 掘削
掘削は、宝物を地下から掘り起こすためのスキルです。
## EXPの入手
シャベルで特定のブロックを掘ることで獲得することができます。
### EXPが獲得できるブロック（発掘素材）
* 草ブロック
* 土
* 粗い土
* ポトゾル
* 砂
* 赤砂
* 粘土ブロック
* 砂利
* 菌糸ブロック
* ソウルサンド
* ソウルソイル
* 雪ブロック
* 泥ブロック
* 泥だらけのマングローブの根
## サブスキル
### ギガドリル（アクティブスキル）
スキルが発動している間は、宝物を発見する確率が3倍になります。また、発掘素材にあるブロックを一瞬で破壊できるようになります。
### 宝物の発掘（パッシブスキル）
発掘素材のブロックをシャベルで採掘すると、確率で宝物がドロップします。それぞれドロップ品によって、何の発掘素材からドロップするかが異なり、確率も異なります。宝物がドロップするとドロップ品に応じたEXPを獲得することができます。
