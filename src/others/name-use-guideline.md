# 名称利用ガイドライン
本ガイドラインは、Nekozouneko Server（以下「当サーバー」）の名称を当サーバー内外で利用する際の厳守事項を定めたものです。皆様に正しい名称の把握をいただくために、ご理解とご協力をお願いいたします。

## 正式名称
当サーバーの正式名称は「Nekozouneko Server」です。

## 非公式SNS上における表記
非公式SNSとは、当サーバーが管理・運営を行っていないSNSアカウントやサーバーのことです。これら非公式SNSを運用する場合、混同を避けるために次のいずれか、又は両方の対応を取ってください。
* 名前（アカウント名・サーバー名など）に「非公式」という言葉を含めてください。
* 説明欄や概要欄などの利用者の目に留まりやすい場所に、公式とは一切関係がない旨を明記してください。
::: tip 非公式SNSとは、例えば次のようなものです。
※あくまで例です。これだけが当てはまるわけではありません。
* Xのアカウント名
* YouTubeのアカウント名
* LINEのオープンチャット
* Discordサーバー
* NEarthの町用Discordサーバー
:::
::: tip 対応例
* 【非公式】Nekozouneko Serverで建築配信！
* このサーバーは、Nekozouneko Server公式とは一切関係ありません。
    * オープンチャットのルール
* この動画は、Nekozouneko Server公式とは一切関係ありません。
    * YouTubeの概要欄
:::

当サーバーの名称に関して、次のいずれかのような利用方法をしてはなりません。

## わからないことがあったら
