# 名称利用ガイドライン
本ガイドラインは、Nekozouneko Server（以下「当サーバー」）の名称を当サーバー内外で利用する際の厳守事項を定めたものです。皆様に正しい名称の把握をいただくために、ご理解とご協力をお願いいたします。

## 非公式SNS上における表記
非公式SNSとは、当サーバーが管理・運営を行っていないSNSアカウントやサーバーのことです。これら非公式SNS上では、**名前（アカウント名・サーバー名など）に「非公式」という表記を行ってください**。
::: tip 例えば、次のようなものです。
※あくまで例です。これだけが当てはまるわけではありません。
* Xのアカウント名
* YouTubeのアカウント名
* LINEのオープンチャット
* Discordサーバー
* NEarthの町用Discordサーバー
:::

## 正式名称の表記方法
当サーバーの正式名称は「Nekozouneko Server」です。当サーバー内外で当サーバーのことを正式に指す場合、次のいずれかを使用してください。
* Nekozouneko Server
* ねこぞ鯖

## 禁止されている行為
当サーバーの名称に関して、次のいずれかのような利用方法をしてはなりません。
1. 公式が管理・運営していると見間違えられる可能性があるような利用方法
::: tip ダメな例と許可される例
※あくまで例です。これだけが当てはまるわけではありません。
* ❌「公式 - ねこぞ鯖民Discordサーバー」
* ❌「Nekozouneko Server 公式チャンネル」
* ❌「ねこぞ鯖サポートセンター」
* ✅「非公式 - みんなで整地をする集い」
* ✅「【非公式】ねこぞ鯖〇〇町Discord」
* ✅「ねこぞ鯖民が紹介する豆知識チャンネル（非公式）」
:::
2. 公序良俗に反するような利用法
3. その他、当サーバーに不利益を与え得るような利用方法

## わからないことがあったら
本ガイドラインでは、一部の例のみを掲載しております。もし、利用が許可されるかご不明な際は、当サーバーへお問い合わせください。
