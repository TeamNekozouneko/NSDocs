# Nekozouneko Server 利用規約
この利用規約（以下「当規約」）はNekozouneko Group（以下「当グループ」）が提供するNekozouneko Server（以下「当サーバー」）でご利用の皆様（以下「プレイヤー」）により良いゲームプレイを提供するため規則、規定、および、条件を定めるものです。プレイヤーの皆様には、当規約に同意いただく必要があります。

## 第一条（適用）
1. 当規約は、当サーバーとプレイヤーとの間の一切の関係に適用されるものとします。
2. 当規約は、当サーバーに参加した時点で同意したものとします。
3. 当グループは、当規約の理解の推進を目的とし、翻訳版および要約版の発行を行うことがあります。ただし、本文面が原文であり、これによりプレイヤーに生じた不利益について、当グループは一切の責任を負いません。
4. 当規約は、民法第五百四十八の二に基づき規定される定型約款であり、プレイヤーは同条第二号における、みなし合意として契約を締結するものとします。

## 第二条（附則）
1. 附則とは、当規約に一定の形で規定できないため、個別で追記した補助的な規約とします。
2. 当規約と附則の間で矛盾が発生した際は、特段の定めなき限り附則を優先します。
3. 各項にあたる特別な各種定め（以下「個別規定」）は、附則と同等のものとします。

## 第三条（禁止事項）
プレイヤーは以下に該当する行為をしてはなりません。
1. Microsoft社が定める EULA（MINECRAFT エンドユーザー ライセンス契約）に違反する行為
2. 当サーバーに対し、DDoS/DoSやハッキング等の攻撃を試みる行為、また、これによって当サーバーへ損害を与える行為
3. 当サーバーに対し、悪意を持ってゲーム上の異常な負荷を与え、当サーバーへ損害等、および、ユーザーエクスペリエンスを阻害する行為
4. Minecraftアカウント等の一つのアカウントを、当グループが提供しているサービスで複数人にわたって利用し、アクセスを試みる行為、または、アクセスをする行為
5. 不正に入手したMinecraftアカウントを利用し、アクセスを試みる行為、または、アクセスをする行為
6. 当サーバーから科された処罰を回避する目的で、同一人物が当該アカウントとは別のアカウントでアクセスを試みる行為、または、アクセスする行為
7. 当サーバーにVPNまたはプロキシ等の通信を別媒体で経由させ、接続元を偽装してアクセスを試みる行為、または、アクセスする行為
8. 当事者間で承諾を得ていないプレイヤーの殺害行為、または、殺害には至らないが、攻撃を行い体力や防具に損害を与える行為
9. 当サーバー内のMinecraftワールド上で、その運営を目的として設置した建造物、および、工作物、もしくは、プレイヤーによって設置された公共的な目的とする建造物、および、工作物（以下「公共物」）の範囲で景観を損ねる行為
10. 当サーバー内のMinecraftワールド上で、他者の所有物（建造物、および、工作物を含みます）であることが明白な範囲において、その所有者の許可を得ずに侵入を繰り返すなど、所有者の不利益になる、もしくは、不快にさせる行為
11. 一般に平等でプレイヤーが使用できるゲーム機能（以下「バニラ機能」）と比べ、優位性を不正に、または、顕著に向上させる追加機能（外部ソフトウェア、ハードウェアの改造等）、もしくは、当サーバーが明示的に許可していないMODを利用し、アクセスする行為
12. バニラ機能において、プレイヤーの実際の操作によって達成できる行為の一部またはすべての動作を自動化する行為
13. Microsoft社が公認していない仕様、もしくは、当サーバーが想定しておらず、明示的に許可していない仕様（以下「バグ」）を利用する行為
14. 当グループに所属する者、および、プレイヤーがオリジナルで作成したスキンや、ユーザー識別子などの固有のものに意図して類似させ、なりすまし、混乱を招く行為
15. 秩序や景観を損ねるおそれがある、公序良俗に反する不適切なスキンや、ユーザー識別子を使用してアクセスする行為
16. 当サーバーの管轄内において、当サーバーが許可していないサーバーやコミュニティ、SNSの宣伝、および、勧誘行為
17. 不必要に連続して類似する内容、もしくは、意味のない内容のチャットを送信する行為（以下「スパム行為」）
18. 過度に暴力的な表現、挑発的な表現、人種差別・性別差別・その他の差別につながる表現、わいせつな表現、その他宗教的、反社会的な内容などを含む他者に不快感を与える発言
19. 他者の個人情報を入手しようと試みたり、自他の個人情報を第三者が閲覧可能な場において言及する行為
20. 当規約に違反したと推測されるプレイヤーに対して、私刑または制裁等の不正な手続きに基づく処罰を科す行為
21. 他者の許可を得ずに、当該プレイヤーの所有物（建造物含む）を損失させる、もしくは、損害を与える行為（以下「荒らし行為」）
22. 当サーバー内におけるアイテム、通貨及びサービス等の価値があるものを法定通貨（リアルマネー）を用いて取引する行為
23. 自他の規約違反を知りながらそれを隠蔽、又は隠蔽を幇助する行為
24. 前各項に準ずる程度の行為
25. 他、当サーバーが不適切であると判断した行為

## 第四条（処罰）
1. 当サーバーは、以下に定める事項のいずれかに該当したプレイヤーに、事前に通知することなく、当サーバーの利用の一部、または、全部を停止する制裁的目的をもって、処罰を科すことができます。
    1. 前項に定めるいずれかの事項に該当した場合
    2. 他者に不利益を与えるなどの迷惑行為の遂行を目的としたコミュニティに所属、もしくは、関与している場合
2. 当サーバーがMinecraftサーバー上で科す処罰は以下の種に限るものとします。
    1. 警告
    2. ミュート（期限付き、もしくは、無期限）
    3. 現接続の遮断（一時的に行う強制ログアウト）
    4. アクセス禁止（期限付き、もしくは、無期限）
    5. 特定のIPアドレスからのアクセス禁止（期限付き、もしくは、無期限）
3. 当サーバーがDiscordサーバー上で科す処罰は以下の種に限るものとします。
    1. 警告
    2. タイムアウト（期限付き）
    3. ミュート（無期限）
    4. 参加禁止（無期限）
4. 当サーバーは、処罰が以下のいずれかに該当する場合、その処罰の解除を行います。
    1. 処罰の制裁執行期限が満了した場合
    2. 証拠不十分、または、嫌疑不十分、または、証拠の提示が困難だと判断した場合
    3. プレイヤーの異議申し立てにより、当規約の違反に該当しないと判断した場合
    4. 他、当サーバーが処罰の解除が適当だと判断した場合

## 第五条（建造物）
1. 当サーバーは、公共物の範囲外において、以下のいずれかに該当する場合、建造物の撤去を行うことができます。
    1. アクセス禁止（無期限）もしくは、特定のIPアドレスからのアクセス禁止（無期限）を科されたプレイヤーが設置した建造物
    2. 最終ログインから2ヶ月以上が経過したプレイヤーの建造物
    3. 当サーバーに損害を与えるおそれがある程度の運用負荷の高い建造物（レッドストーン回路、トラップ装置などを含みます。）
    4. その他、著しく公益に反するおそれがある建造物
2. 当グループのStreamerが配信活動上で使用すると定める特定のサーバーでは、プレイヤーのいかなる建造物の著作権も当グループに帰属するものとします。
3. 当グループは、前項において、著作者人格権は一切行使しないものとします。

## 第六条（ロールバック）
当サーバーは、以下のいずれかの場合において、ある時点から元のある時点へ状態を戻す操作（以下「ロールバック」）を実施することができます。
1. 当サーバーが行ったロールバックが適当に実施されなかった場合
2. 当規約違反の際に他プレイヤーへ被害が生じた場合
3. 当サーバーのシステムの不具合によって損失が生じた場合
4. 前各項に準ずる程度の一般に認められる損失が生じた場合

## 第七条（異議申し立て）
1. プレイヤーは、当サーバーから科された処罰に関して、以下の条件を全て満たす場合に限り、異議の申し立てを申請することができます。
    1. 申請日が処罰の執行日から二ヶ月以内であること
    2. 以前に同一の執行について異議申し立てを申請していないこと
    3. 当該プレイヤー本人からの申請であること
2. 異議申し立てを行う際は、異議として認められる程度の根拠を提示しなければならないものとします。
3. 当サーバーは、当該異議申し立てにおいて、処罰事由、および、それに付随する関連情報について、一切の開示義務を負わないものとします。

## 第八条（方針の決定）
当サーバーは、プレイヤーに事前の通知をもって、当サーバーが構成するゲームコンテンツの変更、追加または廃止することがあり、プレイヤーはこれを承諾するものとします。

## 第九条（対応方針）
当サーバーが、実施する対応（サポートセンター、処罰、ロールバックなどを含みます。）の詳細は別途、当サーバーが定める対応方針に従うものとします。

## 第十条（当規約の変更）
1. 当サーバーは、民法五百四十八条の四に基づき、プレイヤーに事前の通知をもって当規約を変更できるものとします。
2. 当サーバーは、プレイヤーの一般の利益に適合する場合、事前の通知なく当規約を変更できるものとします。

## 第十一条（仲介依頼）
1. プレイヤーは、当サーバーの管轄内における非違反行為による他プレイヤーとの紛争等について、中立的な判断者としての仲介を当サーバーに依頼申請することができます。
2. 当サーバーは、当サーバーが紛争に仲介する合理性があると判断し、かつ、紛争の当事者が仲介に合意した場合、介入することができます。
3. 当サーバーによる仲介が確定した後、当事者は当サーバーの決定に従うものとし、また、これに異議を申し立てることはできません。

## 第十二条（個人情報の取扱い）
当グループは、当サーバーの利用によって取得するプレイヤーの個人情報について、当サーバー「プライバシーポリシー」に従って、適切に取り扱うものとします。

## 第十三条（当サーバーの提供の停止）
当グループは、以下のいずれかの事由があると判断した場合、プレイヤーに事前に通知することなく、当サーバーの全部または一部の提供を停止または中断することができるものとします。
1. 当サーバーにかかるシステムの保守点検または更新を行う場合
2. 地震、落雷、火災、停電を含む、自然災害などの不可抗力により、当サーバーの提供が困難となった場合
3. システムの根幹や中枢部分の破損、または、システムや通信回線等が停止した場合
4. その他、当グループが当サーバーの提供が困難と判断した場合

## 第十四条（免責事項）
1. 当サーバーは、当サーバーが提供するコンテンツに、事実上または法律上における瑕疵（RASIS、情報システム上の欠陥、権利侵害などを含みます。）がないことを明示的にも黙示的にも保証しておりません。
2. 当サーバーは、当サーバーが提供するコンテンツに起因してプレイヤーに生じた損害について、当サーバーの故意または重大な過失を除き、一切の責任を負わないものとします。
3. 当サーバーは、プレイヤーと他プレイヤーとの間、もしくは、第三者との間において生じた取引、契約、連絡、紛争等について一切責任を負わないものとします。
4. 当サーバーがプレイヤーに対して当規約に基づく制裁行為を執行した結果として、当サーバーの利用が停止した場合でも、特定商取引法に基づく売買契約の取り消し等の対応は、別途定める特定商取引法に基づく表記により規定されます。
5. プレイヤーの皆様は数あるコミュニティから当サーバーを自ら選択して利用することができる権利を有しているものと同様に、当サーバーもご利用いただくプレイヤーの皆様を選択することができる権利があるものとします。

## 第十五条（権利義務）
プレイヤーは、利用契約上の地位または当規約に基づく権利もしくは義務を第三者に譲渡し、または担保に供することはできません。

## 第十六条（準拠法および裁判管轄）
1. 当規約の解釈にあたっては、日本法を準拠法とします。
2. 当サーバーが提供するコンテンツおよびサービスに関して紛争が生じた場合には、当グループの通信事業者上の代表者の所在地を管轄する裁判所を第一審の専属的合意管轄とします。
