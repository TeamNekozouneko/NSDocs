# 附則（NEarthサーバー）
この文書（以下「本附則」）は、Nekozouneko Server 利用規約の附則とします。
## 第一条（本文における定義）
|用語|定義|
|-|-|
|Lands|プラグインの一種、主に国家、土地及び戦争の管理を行います。|
|共同体|いわゆるLandsにおける次のものの総称、連邦、国及び帝国|
|集団|いわゆるLandsにおける次のものの総称、集落、村、街、都市及び大都市（ただし、共同体も集団に含まれます。）|
|識別名|集団名及び役職名を含む、Landsのシステム上第三者が閲覧可能な場所に明記する名前全般|
|オーナー|集団のオーナー役割に指定されるプレイヤー|
|所属者|集団に所属するプレイヤー|
|集団内規則|オーナーと各所属者との間に適用される所属条件及び禁止事項等の各種定め|
|約款|同一の定型的な契約内容を不特定多数と締結する契約|
|アースマップ|地球の地理データを一定比率でワールドに再現し、その現実と仮想との繋がりがあることを魅力とするマップ全般|
|敵対機能|Landsプラグインにより、システム的に認められる外交及び戦争関係の機能（集団を敵対としてマークしたり、戦争を実施する機能を含みます。）|
|エリア|Landsプラグインにより、自由な範囲でより小さく分割された区画及び領土全体の空間|
## 第二条（オーナーの責任）
1. [オーナー](#第一条-本文における定義)は、[集団](#第一条-本文における定義)の運営に係る一切の責任を負うものとします。
2. オーナーは、集団内の事情及び[所属者](#第一条-本文における定義)に、当サーバーが定める各種定めを厳守させる義務を負うものとします。
## 第三条（集団内規則）
1. オーナーは、個々の形式に基づき[集団内規則](#第一条-本文における定義)を制定することができるものとします。
2. オーナーは、集団内規則を制定又は改正するとき、当サーバーが定める各種定めに抵触及び矛盾しないものとします。
3. オーナーは、集団内規則が[約款](#第一条-本文における定義)である場合、所属を予定しているプレイヤーに対して集団内規則のすべてを事前に提示するか、必ず当該プレイヤーの同意を得るものとします。
## 第四条（禁止事項）
下記のいずれかに該当する行為をしてはなりません。
1. [Nekozouneko Server 利用規約第三条（禁止事項）](/1/terms#第三条-禁止事項)第16項、第17項、第18項及び第19項に準ずる識別名又はなりすましの意図をもって他集団の識別名と著しく類似する識別名を使用する行為。
2. 同一人物が3つ以上のアカウントを同一の集団に所属させる行為
3. [アースマップ](#第一条-本文における定義)の魅力を著しく低下させる程度にマップを改変する行為。
4. 集団の周辺を囲うように土地を確保するなど、[敵対機能](#第一条-本文における定義)を使用せず他の集団に損害を与える行為。
5. ゲームバランスの設定の不備などによって、異常に高い効率が得られる状態や、ゲームシステム上のバグもしくは不具合など、一般に長期的なゲームシステムを維持する観点に反する状態を知りながら、当サーバーに報告せず、継続的に利用する行為。
## 第五条（処罰及び対応方針）
1. 当サーバーは、下記のいずれかの事項に該当すると判断した集団に対して、オーナーに相当期間定めてなした勧告後も状況が是正されない場合、集団の削除を強制することができるものとします。
    1. [第三条（集団内規則）](#第三条-集団内規則)第2項に該当する集団内規則を制定している場合
    2. [第三条（集団内規則）](#第三条-集団内規則)第3項に基づき、定型約款の正当な手続きを行っていない場合
1. 当サーバーは、[第四条（禁止事項）](#第四条-禁止事項)第3項に該当すると判断した改変行為に対して、その所有者に相当期間定めてなした勧告後も状況が是正されない場合、強制的に再生成又はロールバックをすることができるものとします。
## 第六条（免責事項）
1. 当サーバーは、[第三条（集団内規則）](#第三条-集団内規則)第1項に基づき制定された集団内規則に起因して生じた損害及び紛争等について、一切の責任を負わないものとします。
2. 当サーバーは、集団が[エリア](#第一条-本文における定義)において自ら設定した各種権限に起因して生じた損害について、一切の責任を負わないものとします。
3. 当サーバーは、敵対機能によって生じたアイテムの損失について、一切の責任を負わないものとします。
4. 当サーバーは、Lands以外の媒体により主張された土地について、一切の責任を負わないものとします。
