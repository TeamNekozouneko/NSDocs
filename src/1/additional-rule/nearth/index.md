# 附則（NEarthサーバー）
この文書は Nekozouneko Server 利用規約の附則とします。
## 第一条（本文における定義）
|用語|定義|
|-|-|
|Lands|プラグインの一種、主に国家、土地及び戦争の管理を行います。|
|共同体|いわゆるLandsにおける次のものの総称、連邦、国及び帝国|
|集団|いわゆるLandsにおける次のものの総称、集落、村、街、都市及び大都市（ただし、共同体も集団に含まれます。）|
|名前|集団名及び役職名を含む、Landsのシステム上第三者が閲覧可能な場所に明記する名前全般|
|エリア|集団の領土のうち、自由な範囲でより小さく分割された土地|
|敵対機能|Landsプラグインに標準で搭載され、システム的に認められる外交及び戦争関係の機能（集団を敵対としてマークしたり、戦争を実施する機能を含みます。）|
|アースマップ|地球の地理データを一定比率でワールドに再現し、その現実と仮想のリンクがあることを魅力とするマップ全般|
## 第二条（禁止事項）
1. [Nekozouneko Server 利用規約](https://blog.nekozouneko.net/terms)第三条（禁止事項）第16項、第17項、第18項及び第19項に準ずる程度の内容を名前に使用する行為。
2. 同一人物が3つ以上のアカウントを使用し、同一の集団に所属させる行為。
3. アースマップの魅力を著しく低下させる程度にマップを改変する行為。
4. 集団の周辺を囲うように土地を確保するなど、敵対機能を使用せず他の集団に損害を与える行為。
5. ゲームバランスの設定の不備などによって、異常に高い効率が得られる状態や、ゲームシステム上のバグもしくは不具合など、一般に長期的なゲームシステムを維持する観点に反する状態を知りながら、当サーバーに報告せず、継続的に利用する行為。
