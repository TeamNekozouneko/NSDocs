## 要素内容
### 経済システム
* 単位: G（ゴールド）
* `1G = 1金インゴット`の金本位制を採用
::: warning
金本位制を維持するために下記の制限があります。
* ゾンビピグリンからは金がドロップしません。
* 金装備や金ツールは精錬できません。
* ネザー金鉱石の金塊のドロップ率が大幅に低くなっています。
:::
### 称号
集団や国には、その規模を表すために名前の前に`二つ名`として称号が付きます。
#### 集団の場合
|称号|条件|
|-|-|
|集落|住民が4名未満|
|村|住民が4名以上、かつ、銀行残高が100G以上|
|街|住民が8名以上、かつ、銀行残高が300G以上|
|都市|住民が14名以上、かつ、銀行残高が550G以上|
|大都市|住民が24名以上、かつ、銀行残高が1,000G以上|
#### 国の場合
|称号|条件|
|-|-|
|連邦|所属する集団が3つ未満、または、住民が32名未満、または、銀行残高が3,000G未満|
|国|所属する集団が3つ以上、かつ、住民が32名以上、かつ、銀行残高が3,000G以上|
|帝国|所属する集団が6つ以上、かつ、住民が64名以上、かつ、銀行残高が5,000G以上|
### Landsの費用
* 集団の設立費用: `30G`
* 1チャンクあたりの土地の費用: `5G`
* 1チャンクあたりの維持費: `3G`
#### 土地確保の制限
* 他の集団/国の領土から`8チャンク`以上離れていなければ確保できません。
#### 維持費
維持費の回収間隔: `1週間（7日）`
::: tip
維持費等の日数は、**現実世界の日数です**。ゲーム内日数ではありません。
:::
維持費は**滞納することができません**。支払い予定の維持費が現時点で不足している場合、期限の`2日前`から警告が通知されます。もし、全額支払うことができなかった場合、維持費は払える分だけを支払い、払えなかった分の土地は所有権を失います。（土地の確保が新しい順から失います。）また、**一切土地の維持費が支払えなくなると集団は強制的に解散します**。
#### コマンドの実行費用
* /lands setspawn
  * 初回: `無料`
  * 2回目以降: `10G`
* /lands spawn
  * 所属メンバー: `無料`
  * 外部メンバー: `5G`かつ`60秒のクールダウン`
* /lands rename
  * `50G`かつ`24時間のクールダウン`
#### 非アクティブな集団の解散
集団のリーダー（オーナー）のプレイ時間に応じて、特定の期間サーバーにログインがない場合、自動的にその集団は解散します。しかし、他の所属メンバーがアクティブに残っている場合、**役職が高い順から新たなリーダーに任命されます**。
|プレイ時間|解散されるオフライン期間|
|-|-|
|15分以下|7日以上|
|30分以下|14日以上|
|12時間以下|30日以上|
|24時間以下|60日以上|
|30日以下|80日以上|
|31日以上|365日以上|
::: info
例えば、プレイ時間が25分の場合、14日以上オフラインが続くと、そのプレイヤーがリーダーとなっている集団は解散されます。
:::
## 注意事項
::: danger
集団内のチャット・国内のチャットは、通常外部のプレイヤーからは閲覧できません。しかし、**運営は常にチャットの内容を見ることができ、すべてログとして記録されています**。常識とマナーに気をつけて利用しましょう。
:::
