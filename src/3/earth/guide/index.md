# 遊び方ガイド
Earthサーバーの遊び方について説明します。

## まず入ったら...
大きく2つの選択肢があります。
* どこかの集団に所属する
* 自分で集団を作る

## どこかの集団に所属する
チャットで街や国に入れてもらえるか聞いてみたり、Discordのサーバー内宣伝で積極的にメンバーを募集している街や国に参加してみましょう。
### 集団を知る
どこかの集団に所属することができたら、その場の雰囲気や集団の流れを掴んで協力しましょう。一人で進めることが難しい場合は、周りの人に聞いてみましょう。

## 自分で集団を作る
街や国は元々、プレイヤーの集まりが発展していくことから始まります。Landsプラグインによって土地を取ることができます。その土地にプレイヤーを招き入れることで、発展度に応じて`集落` `村` `街` `都市` `大都市`と呼ばれるようになります。
### 資源や資金を集める
土地を初めて取る場合、土地を宣言するために`30G`必要です。また、1週間間隔で`確保しているチャンク×3G`の維持費がかかります。Earthサーバーは**金本位制**、つまり金がそのまま通貨として使用できます。地下を採掘して鉄や金、ダイヤモンドなどを集めて、集団を作る準備をしましょう。
::: tip
1G = 1金インゴットだよ！土地の作り方は[Lands](/2/lands/)を参照してね。
:::
### プレイヤーを招待する
土地は、その集落に所属しているプレイヤーにつき`5チャンク`分が確保できるチャンク数が増えます。そのため、所属するプレイヤーが多ければ多いほど大きな集落となり、発展もより加速します。所属した時の特典を用意したり、所属するメンバーのモチベーションを維持して、交流関係を深めましょう。
