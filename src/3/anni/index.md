# ANNI
ANNIサーバーは、とある海外のサーバーにあるゲームが元となり生まれました。プレイヤーは通常2チームに分かれ、「ネクサス」というエンドストーンを削られないように守り、逆に敵チームのネクサスを壊して攻めるゲームです。

ネクサスが削られるとチームのポイントが減り、0になると負けとなります。協力して勝利を目指しましょう！

## 特徴
* サーバーが日本にあり低遅延で遊べる
* チャージ式PvPで、連打に自信がなくても楽しめる。
* 本家にはないようなキット

## 遊び方
1. サーバーに入る。
    * ANNIサーバーのNPCをクリックしたり、`/server anni`より参加できます。
2. `/kit`でキットを選ぶ
    * 色々なキットがあるのでとりあえず気になったものを試してみましょう！
3. 資源を集め、フェーズ2以降のネクサスを削るの準備をする。
    * 木こりキットでは木を多くゲットできます。
    * 採掘師では専用のツルハシで鉱石を多く入手できます。
    * エンチャンターでは獲得経験値が2倍になります。
4. ネクサスを削る
    * 失敗しても、すぐに生き返ることができます。色々な削り方を試してみましょう。例えば...
        * ポーションショップ (`/potion-shop`) で透明ポーションのアイテムを揃え、醸造し、透明化のままネクサスを削りに行く
        * 仲間と一緒に突撃（ラッシュ）をする
        * 軽装備でちょっとずつ削る
5. ネクサスを護る。
    * 防衛キットの防衛の心を使うと一時的に相手のネクサスの破壊を遅らせることができます。

これだけわかれば大丈夫！あとは何戦も勝利・敗北を経験し慣れるだけです！
