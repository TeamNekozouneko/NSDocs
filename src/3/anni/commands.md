# コマンド
ANNIにはいくつかの専用のコマンドが用意されています。

| コマンド | 説明 |
| --- | -- |
| `/kit` | キットの選択画面を開きます。好きなキットを選んでみよう。 |
| `/redeploy` | 戦闘状態出ない限り、すぐに死亡します。バグに陥った場合などに利用してください。 （代替: `/suicide`, `/kill`）|
| `/player` | あなたのレベル、ポイントやK/D情報などを表示します。 |
| `/combat-shop` | 戦闘に役立つショップを開きます。（代替: `/cshop`, `/shop-combat`）|
| `/potion-shop` | ポーションの醸造に役立つショップを開きます。（代替: `/pshop`, `/shop-potion`）|
| `/vote <map>` | マップを投票できます。好きなマップに投票してみよう。 |
