# NEarthサーバー
NEarthサーバーは、日本サーバーで**最大級スケール1:326**で地球を再現したアースマップに、建国・経済・戦争のリアルな生活要素を取り入れたMMOサーバーです。
## `サーバーの特徴！`
### 日本史上初の1:326スケールの巨大地球再現マップ
どんなサーバーも1:500に留まっているスケール...Nekozouneko Serverならではの大人数・大規模で楽しめる！
### ライフサーバーに続く軽量設計でラグの心配なし！
大規模なマップを一定間隔で区切り、各地で処理を分ける分散技術により、一つのワールドでリアルタイムに大人数が遊んでも快適！
### 時代はLands！戦争を標準搭載、統合版にも対応済み
従来のTownyにはない有料プラグインの安定感があり、視覚的にもわかりやすい土地管理ができます！
## `サーバールール`
::: tip
このサーバールールは、[附則（NEarthサーバー）](/additional-rule/nearth)の`要約版`とします。
:::
### やってはいけないこと
* 町や国の名前、役職の名前を不適切なものにしないでください。
* 町や国の名前、役職の名前を使ってなりすましをしないでください。
* アカウントは**3個まで**しか入れないでください。（メインアカウントとサブアカウント合わせて）
* アースマップを極度に改変しないでください。
* 戦争機能があるので、戦争機能以外で嫌がらせや攻撃をしないでください。
### 町や国を作った方（オーナー）へ
オーナーは、その集団の**責任者**になります。所属している方が、Nekozouneko Serverのルールに違反していないかしっかり監視してください。

もし、所属者がルールに違反していると思ったら運営に報告してください。所属者が違反した場合、処罰は所属者に科されるため、オーナーが処罰を肩代わりすることはありません。
### 集団内のルール
オーナーは、作った町や国の中で独自のルールを決めることができます。ただし、しっかり所属している方に**ルールのすべてを周知**してください。
### 運営ができること
運営は、このサーバールールに違反していると判断した場合、まず警告機能を通じて所有者やオーナーに連絡します。

十分に対応ができるであろう期間、警告をしても続けている場合、**強制的に町や国を解散させたり、土地を再生成させることがあります**。
