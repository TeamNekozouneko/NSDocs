# 規約改正のお知らせ
* 作成日 `令和7年11月21日`
* 公布日 `令和7年11月21日`
* 施行日 `令和7年11月24日`
## はじめに
この度、より一般的な文言への変更や、現対応に基づいた表記を行うため、[Nekozouneko Server 利用規約](/terms)及び[The Mining附則](https://blog.nekozouneko.net/archives/724)を改正する運びとなりました。その他、表記揺れにつきましても訂正を行っております。
::: danger 注意
細かな変更内容が非常に多いため、本ページでは要点や契約内容に変更がある箇所のみを変更点として掲載しております。詳しい変更後の条文は[こちら](/terms)をご参照ください。
:::
## 改正内容（Nekozouneko Server 利用規約）
1. 参照又は引用する条文について、アンカーリンクを設置いたしました。
    1. 日本法を参照又は引用する場合、デジタル庁が運営する[e-Gov](https://www.e-gov.go.jp)を用いて、該当条文へのアンカーリンクを設置しております。
2. 次の言い回しを変更いたしました。
    1. 変更前「〇〇、または、〜〜」、変更後「〇〇又は〜〜」
    2. 変更前「〇〇、および、〜〜」、変更後「〇〇及び〜〜」
    3. 変更前「〇〇、もしくは、〜〜」、変更後「〇〇又は〜〜」
    4. 変更前「`(`〇〇、または、〜〜`)`、または、⬜︎⬜︎」、変更後「〇〇若しくは〜〜又は⬜︎⬜︎」
    5. 変更前「以下のいずれか」、変更後「次に掲げるいずれか」（同項各号を指す場合）
    6. その他、一般的な法令用語に基づく言い回しに変更しました。
3. 備考のフォーマットを統一いたしました。
4. [第三条（禁止事項）](/terms#第三条-禁止事項)の項番号を整理するため、より近しい条文同士が隣接するように並び替えました。
5. [第四条（処罰）](/terms#第四条-処罰)第2項
    1. 接続時IPアドレスを用いた処罰は、「アクセス禁止」の包含関係であるため、重複しているものとして削除いたしました。
6. [第五条（建造物）](/terms#第五条-建造物)第2項のチーム名を最新のチームに更新いたしました。
    1. 変更前「Streamer」、変更後「Entertainment」
7. [第三条（禁止事項）](/terms#第三条-禁止事項)の各項における建築物及び工作物の範囲に関する定義をより明確に修正いたしました。
    1. 変更前「建築物及び工作物の範囲で...」、変更後「建築物及び工作物の範囲**内**で...」
8. 次の条文を[第三条（禁止事項）](/terms#第三条-禁止事項)に追加いたしました。
```
ゲームバランスの設定の不備などによって、異常に高い効率が得られる状態及びゲームシステム上のバグ又は不具合などを含む、一般に長期的なゲームシステムを維持する観点に反する状態を知りながら、当サーバーに報告せず、継続的に利用する行為
```
9. [第三条（禁止事項）](/terms#第三条-禁止事項)第17項（旧18項）を次の通りに改正いたしました。
    1. 変更後「暴力的な内容、挑発的な内容、人種差別若しくは性別差別その他の差別につながる内容、わいせつな内容、**政治的な内容**、宗教的な内容又は反社会的な内容などを含む他者に不快感を与える発言**及び公序良俗に反する発言**」
    2. 政治的な内容：あくまで「他者に不快感を与える発言」の例を列挙するものとして追加いたしました。新たに禁止された条項ではありません。
