# Nekozouneko Community
Nekozouneko Communityは、**約7,500名以上**（令和8年1月8日時点）が参加するDiscordコミュニティサーバーです。Nekozouneko ServerやTeam Nekozounekoに関するお知らせがある他、利用者との交流ができるコミュニティとなっています。

また、サポートセンターやTicketによるお問い合わせが可能です。
## 参加方法
::: warning 利用する前の注意事項
Nekozouneko Communityの利用にあたって、[Nekozouneko Server 利用規約](/terms)が適用されます。
:::
### `① 次のリンクをクリックします。`
[https://discord.com/servers/nekozouneko-community-896668963709255680](https://discord.com/servers/nekozouneko-community-896668963709255680)
### `② 画像の赤枠「Join Server」をクリックします。`
![](/images/general/discord-join-button.png)
### `③ 表示される画面に従って参加完了！`

## サポートセンターについて
::: danger 利用する前の注意事項
サポートセンターには、当サーバーが定める[サポートセンター投稿ガイドライン](https://discord.com/channels/896668963709255680/1342872859067945101)があります。**これらを必ず確認してからご利用をお願いいたします。**

ガイドライン違反となる行為が確認され次第、所定の対応を実施させていただく場合があります。
:::
サポートセンターは、次のような場合に当サーバーへお問い合わせを行う場所です。ここは、**他のユーザーからも対応の内容が閲覧できますので、個人情報などが含まれるお問い合わせは一切送信しないでください。**
* 荒らし行為を受けたとき
* 不正行為を発見したとき
* 不適切な発言を受けたとき
* バグや不具合を発見したとき
* その他、基本的なお問い合わせがあるとき
### `投稿方法`
「[サポートを投稿](https://discord.com/channels/896668963709255680/1330403295843455128)」チャンネルから、ご希望の対応を選択してください。表示されるフォームに従って、必須事項を入力し「送信」ボタンを押すことで専用のスレッドが作成されます。
![](/images/general/discord-support-interaction.png)

## Ticketについて
::: danger 利用する前の注意事項
チケットを作成する際、**同チャンネルに記載の注意事項を必ず確認してからご利用をお願いいたします。**
:::
Ticketは、次のような場合に当サーバーへお問い合わせを行う場所です。ここは、**作成者とモデレーター以外は閲覧することができません。**
* 第三者から閲覧されたくないような内容があるとき
* 処罰に関する異議申し立てをするとき
* VPN又はProxyの間違った検出を受けたとき
* その他、サポートセンターではお問い合わせが難しいとき
### `投稿方法`
「[お問い合わせ](https://discord.com/channels/896668963709255680/1019929711461273660)」チャンネルから、「チケットを作成」のボタンを押してください。「Ticketsカテゴリ」内に「ticket-〇〇〇〇」という専用のチャンネルが作成されますので、そのチャンネルに用件を入力して対応をお待ちください。

また、要件が解決しましたら専用のチャンネルの最上部にある「チケットを閉じる」ボタンを押して、チケットを閉じてください。
